# Portable Install Data Layout

This folder is the portable data root used by DefectStudio during development.

- `app/config` stores application defaults shipped with the portable install.
- `app/assets` stores bundled shared assets.
- `app/themes` stores reusable UI appearance themes.
- `app/layouts` stores bundled ImGui layout presets.
- `users/default/config` stores single-user runtime configuration.
- `users/default/config/profiles` stores named configuration profiles.
- `users/default/layouts` stores user ImGui layout files.
- `users/default/exports` stores exported config/profile bundles.
- `projects` stores movable project folders.
- `projects/_template/.defectstudio` shows the intended per-project metadata layout.
