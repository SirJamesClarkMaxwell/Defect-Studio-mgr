# DefectsStudio – Detailed TODO v2

> **Note:** This is a new version of the application written from scratch. Architecture: Modular Domain Monolith, C++23, OpenGL 4.3+, pybind11::embed. 
> The previous beta code serves as **inspiration and reference**, not a base for direct migration. Detailed architecture description in DefectsStudio_Architecture.md.

***

## Project Work Rules

- One feature branch per task: `task/NN-short-name`
- Merge to main only after full Debug + Release build
- Documentation in mdBook in docs/ – updated together with code
- Code generated by AI (Copilot/Codex) is subject to active review before merge
- Large .cpp files (~500 lines) are not allowed – break them into modules/components
- Exception-free zone in rendering paths – document the boundary explicitly
